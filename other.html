PROJECT 1

Exploded 3D Playstation

A 3D Modeled Playstation, from start to finish. Created with grid paper, Adobe Illustrator, Autocad, Cinema4D, and a lot of time.

My first project was a Plannet Parenthood iconography and menu design task

Creating the 3D Exploded Playstation 1 meant physically taking apart a PS1, measuring each piece, and creating what is called an "Orthographic Projection" which is a fancy way of saying the top/sides/front in a linear view.

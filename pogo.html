Pogo Games

We were given the opportunity to take a look at elder user groups and video games while helping to port Pogo's games from an Flash to HTML5. I identified opportunities for the games to better accomodate the long-tail user-base in areas including accessibility, wayfinding principles rather than memory-based principles, and opportunities for fun mental-building activities within games. I was even given the chance to do a little bit of game design!

Check out Pogo and play some games!

The Problem

Pogo.com launched in September of 1999 and has maintained the same user-base for much of that time. The needs of the Pogo users have changed as they have aged, the games and website have remained the same.

My Role

UX Designer, UI Designer, Visual & Interaction Designer, Game Designer (informal), User Researcher

EA Mobile, Charlottetown PEI

2017 - 2018

User Research

We were lucky enough to gain access to years of user data in the form of surveys, feedback forms, forum posts, and troubleshooting information. The UX improvement concepts and simplified user personas came together easily. We were able to use these concepts as guiding principles.

"These users do not know what do to when things break."

"Not concerned with leaderboards or competition, plays games for fun."

"Preference for browser-based and app is nearly split."

"Interested in travel and culture."

"Keeps up with all of the Pogo events."

"Low vision, arthritis, and cognitive decline makes new games hard to pick up."

"Not concerned with leaderboards or competition, plays games for fun."

These points informed the interface elements.




Benchmark Research

Knowing the struggle of elder users with technology I looked at products they already use, like Shoppers Drug Mart's website and perscription programs, as well as benchmark online platforms like Big Fish and Kongregate. I shared my research with Game Designers, Producers, and Engineers in kickoff meetings and via online collaborative documents.

Overwhelming feedback and research dictated minimal changes to the UI.

This user group will read everything we put in front of them (a first in my experience!).

Do not assume prior knowledge, use clear language and never technical jargon.

Vision and motor control issues can be countered with UI layouts.

Cognitive decline can be countered using intuitive wayfinding techniques.

Playing games can ultimately HELP users combat cognitive decline!

This information dictated the information hierarchy.




Deliverables & Implementation

Working with existing content made UI layout changes to the website and games an interesting challenge, every change was carefully reviewed and revised.

Advocated for the user in all areas.

Created and collaborated on mind-mapping UX improvements.

Generated annotated wireframe documents.

Created flow diagrams for web and mobile applications.

Modified screenshots and assets to support user centred design thinking.
